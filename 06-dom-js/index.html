  13. Former Netcapers release Mozilla Browser 1998
  14. Mozilla releases Firefox 2004
  15. Apple releases Safari with Webkit fork of 2003
  16. Google releases Chrome with Blink a Webkit fork 2008
  17. ES4 Abandoned, ES3.1 patch proposal released as ES5 2008
  18. All new ES versions take their release year as their numeral: ES2016 starting 2016
  19. ES6 released and yearly release schedule adopted 2015

The area of influence... bounded by curly braces.

Variables defined in a curly brace scope cannot be accessed outside of that scope, unless they are hoisted.

Variables defined above/outside of the scope can be accessed within a nested scope.

The root level (i.e. no curly braces) is the "global" scope.

More Definition on Functions

  • A function is fundamental building block in any Javascript app
  • It's a procedure that executes a set of statements when invoked
  • A procedure is a a set of serial instructions expressed as code
  • Any amount of arbitrary code can be written within a function block as the procedure
  • The procedures might calculate a value, modify the DOM or communicate with a server, etc..

Callbacks & Anonymous Functions

A callback is a function that is not invoked on the window load, but it is invoked by some kind of trigger (like an event) later on

// Using an anonymous function as a Callback
var months = ['March', '', 'Jan', '', 'Feb']
months.sort((a, b) => Boolean(b) - 1)
// ['March', 'Jan', 'Feb', '', '']

// Using a named function as a Callback
function sortEmpties(a, b) {
  return Number(Boolean(b)) - 1
}
months.sort(sortEmpties)
